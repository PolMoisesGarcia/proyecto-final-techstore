# TechStore

## Descripción

TechStore es una página web ficticia de una tienda de productos tecnológicos.

El proyecto fue desarrollado como pre-entrega para el curso de Front-End JS de Talento Tech.

El sitio incluye una estructura HTML semántica, estilos con CSS externo, navegación interna, sección de productos, reseñas, contenido multimedia y formulario de contacto.

## Tecnologías utilizadas

- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Grid
- Media Queries
- Git y GitHub
- GitHub Pages

## Secciones del sitio

- Inicio
- Productos
- Reseñas
- Multimedia
- Contacto

## Funcionalidades principales

- Navegación interna mediante enlaces a secciones.
- Cards de productos organizadas con Flexbox.
- Reseñas organizadas con CSS Grid.
- Formulario de contacto con campos para nombre, correo electrónico y mensaje.
- Diseño responsive adaptable a distintos tamaños de pantalla.
- Integración de contenido multimedia mediante iframe.
